A terminal-based version of the classic Chopsticks hand game, featuring a Minimax AI with alpha-beta pruning.
This project provides a complete almost playable game with move generation, game-state evaluation, and AI decision logic.

---

## Features

- **Complete Gameplay**: Play by "attacking (\"pass\")" or "splitting (\"stay\")" exactly like the traditional game.
- **Minimax AI with Alpha-Beta Pruning**: The AI evaluates future outcomes and chooses the strongest possible move.
- **Flexible Move Generator**: Includes all legal attacks and splits.
- **Pure Python**: No external libraries required.
